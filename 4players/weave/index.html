acodcha's StarCraft Maps

Weave

Four players are squeezed onto a tiny-sized 64x64 map! The teams consist of the north and east players versus the south and west players. Each player's base features a protected corner expansion. A central hub links all players' bases, though neutral structures must be destroyed to gain access to this hub. The distance to your opponents' bases is very short, so be prepared for quick aggression! Control of the central hub is key!

Select the "Use Map Settings" game type for a 2vs2 team game or for a cooperative game against computers, or the "Free For All" game type for a 4-player free-for-all game.

Compatible with both StarCraft: Brood War and StarCraft: Remastered.
