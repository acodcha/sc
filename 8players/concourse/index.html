Concourse

Eight players compete on a medium-sized 128x128 map. The teams consist of the four north players versus the four south players. Each team features a large high-ground base shared by its four players. Each shared base features a set of sheltered rear expansions along with more exposed forward expansions on medium ground. Each forward expansion leads down to a massive central low-ground pit that features numerous additional expansions; some of these expansions border your team's medium-ground expansions and are more defensible while others are more exposed. The extensive central low-ground pit also features a few medium-ground platforms; control of these platforms and of their surroundings is key!

Select the "Use Map Settings" game type for a 4vs4 team game or for a cooperative game against computers.

Compatible with both StarCraft: Brood War and StarCraft: Remastered.
